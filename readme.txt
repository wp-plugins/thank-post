The plugin is fairly only shows another approach to gratitude the author for the work he has put onto the post.

== Description ==

Thank Post bascially adds a text that you pick and design and afterwards previews it after or before the post ..depending on the how the admin wishes to show the thank ability.

Chaning the default "Thank Post" Design can be done in the settings-> ThankPost submenu. There you can simply tweak around the options.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the whole zip file.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Than simply if you would like to change the default style of the "ThankPost" location and style you go to options.
